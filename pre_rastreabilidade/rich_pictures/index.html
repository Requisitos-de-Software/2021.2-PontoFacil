    1. Introdução

    Um Rich Picture é um desenho que visa representar os principais elementos de um problema para apoiar e proporcionar uma melhor compreensão por parte da equipe de desenvolvimento.

    2. Metodologia

    Primeiramente, foi realizado um rich picture em papel para ilustrar uma idéia geral do funcionamento do aplicativo e a partir disso foram realizadas mais duas versões aprimoradas utilizando a ferramenta Figma. Além disso, os dois autores se encontraram pela ferramenta Discord onde utilizaram o aplicativo simulando um usuário em diversas situações para a elaboração do documento. A reunião não foi garavada.
